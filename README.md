# Seeker

A modern dual-pane file manager for macOS, built with SwiftUI.

## Features

- **Dual-pane layout** with independent navigation
- **Tabbed browsing** — multiple tabs per pane
- **Three view modes** — List, Icons, and Column browser
- **Sidebar** with quick access to Favorites and Locations
- **File operations** — Copy, Move, Rename, Delete, Duplicate, New Folder/File
- **Cross-pane operations** — Copy/Move files between panes
- **Drag and drop** with Option-key to copy
- **Quick Look** preview
- **Search/Filter** in toolbar and per-pane
- **Sync browsing** — navigate both panes simultaneously
- **Keyboard shortcuts** for common actions
- **Native macOS icons** via NSWorkspace

## Requirements

- macOS 26.0+
- Swift 6.2+

## Build & Run

```bash
swift build
swift run
```
